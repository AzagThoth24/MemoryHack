# MemoryHack For Warcraft III

## 什么是MemoryHack
    使用return bug和union bug解锁jass读写内存的权限，达到运行机器码加载dll的目的。

## 支持的版本
    - 1.27a

## 预计支持的版本
    - 1.26a
    - 1.24e

## 如何导入MemoryHack库
    - 选择给定的编辑器版本(1.31.8或者1.32.13)，打开MemoryHack库.w3x，将其中的代码复制进你的地图
    - 将MemHack.dll导入你的地图，并修改路径为MemHack.dll，即删除前面的war3mapImported\
